# rent-finder

Personal automated rental listing pipeline for Toronto Facebook Marketplace.

> **Setup instructions, cookie export procedure, and usage guide will be added in Milestone 11.**

## Quick Start (placeholder)

```bash
pip install -r requirements.txt
playwright install chromium
cp .env.example .env
# Edit .env with your API keys and Telegram bot token
python -m rent_finder.main --help
```
